# Alokah Superapp Backend

### Alokah Superapp Backend is a Node.js application that provides a RESTful API for managing users, hotels, and inventory.

## Installation

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

```bash
npm install
```

## Usage

```bash
npm run dev
```
